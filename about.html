关于本站

个人开发者的第一个网站项目

这个网站是什么

这是一个由个人开发者手工打造的《地平线：零之曙光》与《地平线：西之绝境》粉丝攻略站点。

网站内容涵盖武器图鉴、机械兽弱点分析、剧情流程指引和配装推荐，希望能帮助每一位新猎人顺利踏上冒险之旅。

技术栈

这个网站使用最基础的网页技术构建：

  • HTML5 — 网页结构
  • CSS3 — 视觉样式（暗黑主题，响应式布局）
  • JavaScript — 交互效果（导航菜单、回到顶部等）
  • GitHub Pages — 免费托管与全球 CDN 加速

没有使用任何框架或第三方库，纯手工编写每一行代码。

开发故事

这是站长的第一个网站项目。从零开始学习网页开发，在 AI 辅助下完成了设计、编码和部署的全流程。

如果你也是刚入门的新手，希望这个项目能给你一些启发：做出一个能跑的网站，没有想象中那么难。
